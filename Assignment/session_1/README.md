# Insta Clone

This is a Flutter project that aims to recreate core Instagram features using Flutter and Dart.

## Project Structure

### lib/

The `lib/` folder contains the main Dart source code for the application. This is where widgets, screens, business logic, and application state are typically implemented.

### android/

The `android/` folder contains Android-specific configuration and native code. It includes files such as `AndroidManifest.xml`, Gradle build scripts, and settings required to build and run the app on Android devices.

### ios/

The `ios/` folder contains iOS-specific configuration and native code. It includes files such as `Info.plist`, Xcode project settings, and other resources needed to build and run the app on iPhones and iPads.

### test/

The `test/` folder contains automated tests for the Flutter application. These tests help verify that widgets, functions, and application logic work correctly and continue to behave as expected after changes.
